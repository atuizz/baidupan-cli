BaiduPan CLI API

BaiduPan CLI API, docs, and test console

Third-party CLI and API wrapper built on the official bdpan CLI. One account maps to one config directory. The main supported workspace is /apps/bdpan.

  • Built on the official bdpan CLI. No cookie grabbing or browser reverse engineering.
  • One account maps to one independent config directory.

Protocol overview

Protocol Description Entry
REST JSON Primary API with a unified response envelope. /api/v1/*
Multipart Browser or SDK file uploads. /api/v1/files/browser-upload-share
Compat Query Compatibility layer for query and form style integrations. /compat/baidu/*
OpenAPI Standard Swagger and ReDoc output. /docs / /redoc
